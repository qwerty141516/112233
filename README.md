$ pwd
Print Working Directory
현재 폴더 표시

$ cd
Change Directory

$ mkdir
Make Directory

$ ls
Fire or folder list
-l : 파일의 상세정보
-a : 숨김 파일 표시
-al : 위 두개를 조합


파일 생성

$ touch fname


빈파일 fname 생성

결과 표시
$ echo git bash
$ echo 'print()'

$ cat fname

$ cp a b
파일 a를 b로 복사

$ mv f1 f2
파일 f1을 f2로 이름 수정

$ rm fname
파일 fname 삭제

$ rm -rf dname

-f 강제로 파일이나 디렉토리를 삭제
-r 디렉토리 내부의 모든 내용 삭제
